## About

The goal of this repository is to provide clear and beginner-friendly explanations of PicoCTF Web challenges.  
These writeups help in understanding:

- Basic web vulnerabilities
- How to analyze requests and responses
- How to exploit common web issues
- How to think through CTF problems step-by-step

## Topics Covered

- Input validation issues
- Source code inspection
- Cookie manipulation
- Directory traversal
- Basic XSS
- Encoding/decoding
- Hidden parameters
- Web logic flaws
- HTTP request tampering

## Tools Used

- Python scripts

- curl / wget
- CyberChef
